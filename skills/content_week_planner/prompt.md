# Workflow: วางแผนคอนเทนต์รายสัปดาห์

ทำตามขั้นตอนนี้ตามลำดับ:

1. เช็คข้อมูลที่มีก่อน: ถ้าในโฟลเดอร์งานมีไฟล์แผนเก่า/ข้อมูลแบรนด์ (ดูด้วย `list_files`)
   ให้อ่านมาใช้ก่อน จะได้ไม่ถามซ้ำ
2. ถ้ายังขาดข้อมูล ให้ถามผู้ใช้ **ครั้งเดียวจบ**: ธุรกิจ/เพจเกี่ยวกับอะไร,
   กลุ่มเป้าหมาย, แพลตฟอร์ม (Facebook/TikTok/IG), โพสต์กี่ครั้ง/สัปดาห์
3. สร้างแผน 7 วันเป็นตาราง: วัน | ประเภทคอนเทนต์ | หัวข้อ | hook เปิดโพสต์ (1 ประโยค) | CTA
   - สลับประเภทให้หลากหลาย: ให้ความรู้ / เล่าเรื่อง-รีวิว / ขาย / มีส่วนร่วม (ถาม-โพล)
   - hook ต้องเจาะกลุ่มเป้าหมายที่ระบุ ไม่ใช่ประโยคกลางๆ
4. เสนอบันทึกแผนด้วย `write_file` ชื่อ `แผนคอนเทนต์-<สัปดาห์ที่/วันที่>.md`
5. ปิดท้าย: ถามว่าจะให้ร่างโพสต์เต็มของวันไหนก่อน

กติกา: ใช้ภาษาเดียวกับกลุ่มเป้าหมาย อย่าเสนอหัวข้อที่ต้องใช้ข้อมูลที่ผู้ใช้ไม่มี
